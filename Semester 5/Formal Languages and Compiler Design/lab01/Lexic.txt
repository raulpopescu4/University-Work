Alphabet:

a. Upper (A-Z) and lower case letters (a-z) of the English alphabet

b. Underline character '_';

c. Decimal digits (0-9);

Lexic:

a. Special symbols, representing:

   - operators + - * / := < <= = >=

   - separators [ ] { } : ; space

   - reserved words:

       char const else if int string program read then var while write from to def  for print

b. Identifiers:

   - A sequence of letters and digits, such that the first character is a letter; the rule is:

     identifier ::= letter | letter{letter}{digit}

     letter ::= "A" | "B" | ... | "Z"

     digit ::= "1" | ... | "9"

c. Constants:

1. Integer - rule:

    noconst := "+"no | "-"no | no

    no := digit{no}

2. Character:

    character := 'letter' | 'digit'

3. String:

    constchar := "string"

    string := char{string}

    char := letter | digit
